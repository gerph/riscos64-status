# Documentation updates for 64-bit interfaces

In case anyone else is working on RISC OS 64 at the moment, the interfaces implemented by RISC OS Pyromaniac have been updated in the [PRM documentation](https://pyromaniac.riscos.online/pyromaniac/prm/overview/riscos64-interfaces.html) to include more information about the module entry information and code variables.

This is in addition to the notes present in the [simple binaries repository](https://github.com/gerph/riscos64-simple-binaries/) which will be incorporated into the documentation shortly.

There's also an updated `Sys$Arch` variable, which I've added and requested an allocation for, to enable differentiating the versions of the OS.
