# Initial announcement of 64-bit work and bounty claim

Hiya,

Last year, I demonstrated my AArch64-capable version of RISC OS. This was part of my presentation about the ways in which RISC OS needs to change to accommodate modern hardware. If you didn't catch it, you can find the recording of this presentation on [my site](https://presentations.riscos.online/64bit/index.html).

RISC OS Open have begun its funding for the development of an AArch64-capable version of RISC OS. Having already produced such a system, with the necessary libraries, build system and documentation, I have submitted a claim for part of this bounty.  I've written an open letter to the community with a link to the claim.

Please see: http://riscos.online/rool-bounties/64bit-dev-letter.pdf

Wheeeeee - 64-bit is the future!
